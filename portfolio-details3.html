Project Overview

Designed and developed an autonomous robot integrated with IMU, Arduino camera, and ultrasonic sensors for sophisticated real-time object recognition and obstacle avoidance. This project highlights the seamless fusion of hardware components for effective environmental interaction and navigation.

Key Features

  • Engineered the robot for advanced real-time object recognition and obstacle avoidance, utilizing IMU, Arduino, Raspi cam, and ultrasonic sensors.
  • Achieved precise pick and place operations with a 78% success rate in identifying and handling blue, green, and red blocks within a challenging 10-minute timeframe.

Technologies Used

  • IMU for motion sensing
  • RaspberryPi Cam for visual input
  • Ultrasonic sensors for distance measurement and obstacle detection
