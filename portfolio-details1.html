Project Overview

TerpBot is a leader-follower robot designed to autonomously track and follow a leader while avoiding dynamic obstacles. This project integrates advanced perception techniques using the YOLOv5 object detection model and MiDaS monocular depth estimation to enable robust leader detection and obstacle avoidance.

Key Features

  • Leader Detection: Utilizes YOLOv5 for real-time, multi-class object detection, ensuring accurate tracking of the designated leader.
  • Depth Estimation: Implements MiDaS for monocular depth estimation, enhancing the robot's ability to perceive its surroundings and navigate autonomously.
  • Obstacle Avoidance: Combines the outputs of the detection and depth models to formulate an effective obstacle avoidance strategy, ensuring safe and efficient navigation.
  • Hardware and Control: Features a custom-built platform using Raspberry Pi and Arduino, optimized with an Anti-Windup PID controller for precise wheel control and trajectory tracking.
